# ATM-dispencer
This project gives the count of number of denominations will get for the entered amount. 
For example: If customer entered 12717 rupees, 

## output will be like this: 
0 notes of Rs 1
1 notes of Rs 2
1 notes of Rs 5
1 notes of Rs 10
0 notes of Rs 20
0 notes of Rs 50
0 notes of Rs 100
1 notes of Rs 200
1 notes of Rs 500
6 notes of Rs 2000
Total note dispensed: 11
